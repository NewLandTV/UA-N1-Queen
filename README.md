# 소개

**UA-N1-Queen**모델은 [Ai project](https://github.com/NewLandTV/Ai-project)에 쓰이는 대규모 언어 모델(Large Language Model, LLM)입니다.

## 개발 동기

&nbsp;&nbsp;&nbsp;&nbsp;기존 언어 모델은 너무 방대한 이유로 방송용으로 적합하지 않아, 이런 까닭으로 누구나 이르고자 하는 소망이 있어도 결국 실현하지 못할 사람이 많아, 이를 위해 답답한 마음을 해결하고자 새로운 방송용 언어 모델을 만들어 누구나 쉽게 방송에 편히 쓰이고자 할 나름이다.

## 목표

&nbsp;&nbsp;&nbsp;&nbsp;**나만의 인공지능**을 만들고, **직접 방송**하며 **가상 버츄얼 유튜버**의 기본이 될 수 있게 도와줍니다.
